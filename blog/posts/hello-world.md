# 你好，世界 — 欢迎来到 NeedHub

欢迎来到 **NeedHub**！这是我在互联网上的个人空间，用来分享关于计算机、编程和各种实用工具的学习心得。

## 为什么叫 NeedHub？

我相信学习的最好方式就是**把学到的东西分享出去**。每当我解决了一个棘手的问题、发现了一款优秀的软件、或者读完了一篇有趣的论文，我都想把它记录下来——不仅是为了自己，也是为了所有可能用得上的读者。

## 这里有什么

本博客主要涵盖三个方向：

### 技术分享 💻

深入探讨编程概念、算法、系统设计，以及计算机科学领域的最新进展。无论是解释区块链共识协议的工作原理，还是拆解一个复杂算法，我都力求让技术话题变得通俗易懂。

### 软件推荐 🚀

我一直在寻找能提升效率的工具。这里会分享我的精选发现——终端工具、IDE 插件、浏览器扩展、桌面应用——一切能让开发者生活更美好的东西。

### 学习笔记 📝

来自课程、书籍、论文和动手实践的结构化笔记。从密码学到云计算，从操作系统到机器学习——只要我在学，就在记录。

## 技术栈

本站使用以下技术搭建：

- **VitePress** — 基于 Vue 3 的极速静态站点生成器
- **GitHub Pages** — 免费、可靠的托管服务
- **GitHub Actions** — 每次推送自动部署
